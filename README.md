# FavLinks (React Project)

A simple React app to save your favorite website links using React Hooks, Props, and Tailwind CSS.

## Setup

```bash
git clone https://github.com/kanok99/favlinks-react.git
cd favlinks-react
npm install
npm start
```

## Features
- Add and delete favorite website links.
- State managed in parent component (LinkContainer).
- Responsive, clean UI using Tailwind CSS.
- Functional components + React Hooks.

## Extra Credit
Styled with Tailwind CSS (light blue theme).
